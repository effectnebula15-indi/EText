# EText

**Effect Text Editor** — a simple, contrast-first text editor for Android,
built for `.txt`, `.md` and `.conf` files with a JetBrains-inspired UI.

## Features

- **Plain-text editing** for `.txt`, `.md`, `.conf` (and any `text/*` file).
- **Tabbed multi-file editing**, Windows 11 Notepad-style — open several files
  at once (multi-select in the picker), switch between them via the tab strip,
  add tabs with **+**, close with **×**. Unsaved tabs show a ● dot and prompt
  before closing.
- **Line-number gutter** with a current-line highlight, drawn in sync with
  the text while scrolling — JetBrains-editor style.
- **Monospaced** editing surface with a caret tinted to the active theme.
- **Open / Save / Save As** via the Storage Access Framework (no broad storage
  permissions required) — also opens files sent from other apps (`VIEW`/`EDIT`).
- **Word-wrap toggle** and a live status bar (tab index, `Ln`, `Col`, line/char
  counts).
- **Seven high-contrast themes**, remembered between launches.

## Themes

Base:

| Theme | Canvas | Accent |
|-------|--------|--------|
| Light | white | blue |
| Dark  | `#1E1F22` | blue |

Extra high-contrast dark schemes:

| Theme | Canvas | Accent |
|-------|--------|--------|
| Shadow      | pure black | cool grey |
| Dark-Red    | near-black red | `#FF3B3B` |
| Dark-Purple | near-black purple | `#B388FF` |
| Dark-Pink   | near-black magenta | `#FF45A0` |
| Dark-Yellow | near-black amber | `#FFD60A` |

Switch via **⋮ → Theme**.

## Install

Grab `EText-1.0.0-beta1.apk` from the repository root and sideload it
(Settings → *Install unknown apps*). Minimum Android 7.0 (API 24).

## Build from source

Requires the Android SDK (platform 34, build-tools 34) and JDK 17+.

```bash
# Debug build
./gradlew :app:assembleDebug
#   -> app/build/outputs/apk/debug/app-debug.apk

# Signed release build (expects etext-release.keystore at the repo root)
./gradlew :app:assembleRelease
#   -> app/build/outputs/apk/release/app-release.apk
```

The release signing keystore is **not** committed. Generate your own:

```bash
keytool -genkeypair -v -keystore etext-release.keystore -alias etext \
  -keyalg RSA -keysize 2048 -validity 10000
```

Override credentials with `-PETEXT_STORE_PASSWORD=…`, `-PETEXT_KEY_ALIAS=…`,
`-PETEXT_KEY_PASSWORD=…` (defaults are used otherwise).

## Project layout

```
app/src/main/
  java/com/etext/editor/
    MainActivity.kt     # UI, tabs, file I/O (SAF), theming, status bar
    Document.kt         # per-tab open-file model
    CodeEditText.kt     # monospaced editor + line-number gutter view
    EditorTheme.kt      # the seven color schemes
  res/                  # layouts, menu, vector icons, generated launcher icons
tools/make_icon.py      # regenerates the launcher icon set
```

## Icon

The launcher icon is generated by `tools/make_icon.py` (a gradient squircle
with a bold *E* and a text caret) — run `python3 tools/make_icon.py` to
regenerate all density buckets and the adaptive-icon layers.
